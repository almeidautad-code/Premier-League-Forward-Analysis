# 📊 Premier League — Forward Statistical Analysis & Dashboard

## 📌 Overview
This project performs a **statistical analysis** of Premier League forwards using **Z-scores and T-scores** to normalize and compare player performance across key metrics. It includes a **Forward Profile Comparison Dashboard** built in Excel, allowing side-by-side evaluation of attacking players based on standardized data.

## 🧮 Methodology
1. **Data Collection** — Premier League forward statistics sourced and compiled
2. **Z-score Normalization** — Each metric standardized relative to the mean and standard deviation of all players
3. **T-score Conversion** — Z-scores converted to T-scores (mean=50, SD=10) for easier interpretation
4. **Dashboard** — Final scores visualized in a comparison dashboard

> A T-score above 50 means the player performs **above average** in that metric. Below 50 means **below average**.
